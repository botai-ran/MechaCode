# ADR 0005：本地数据库、凭据库和日志归属

状态：阶段 0 冻结方向，阶段 4 完整实施。

## 背景

后续会引入 SQLite、Run journal、OS 凭据库、日志和诊断包。必须在安全边界上先明确归属。

## 决策

- SQLite 归属 `apps/desktop/src-tauri`，前端通过受控 command 访问。
- API Key 存入 OS 凭据库或等价机制，SQLite 只保存引用标识。
- 日志、工具输出和诊断包默认脱敏，且有大小和保留上限。
- 诊断包默认不包含聊天正文、文件内容和密钥；生成前展示内容清单。

## 影响

- `packages/shared` 不承载安全策略或协议类型。
- 凭据读取只能发生在 provider 配置层或未来凭据 service，不进入工具环境。
