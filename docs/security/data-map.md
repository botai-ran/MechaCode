# 阶段 0 数据地图

状态：阶段 0 基线，后续新增持久化、诊断包或凭据入口时必须同步更新。

| 数据 | 来源 | 当前存储 | 流向 | 保留与删除 | 阶段 0 约束 |
| --- | --- | --- | --- | --- | --- |
| 用户消息 | Desktop UI | 前端内存状态 | Runtime、Provider 请求 | 应用进程退出后丢失 | 不包含工具拒绝原因以外的秘密 |
| 文件内容 | 工作区文件工具 | 不持久化 | 工具结果、模型上下文 | 单次 Run 内存 | 敏感路径默认拒绝；结果脱敏 |
| 工具输出 | Tools | 不持久化 | Runtime 事件、UI 展示、模型 tool 消息 | 单次 Run 内存 | stdout/stderr 和读取内容统一脱敏 |
| API Key | `.env` 或环境变量，后续 OS 凭据库 | 当前仅 provider 配置层读取 | Provider SDK | 不写入聊天、工具结果、日志或诊断 | 工具和子进程不得读取或继承 |
| 子进程环境 | Runtime/Tools 父进程 | 子进程内存 | `git` 或命令工具 | 进程结束即释放 | 显式 allowlist；移除 key、token、proxy、Git/SSH 凭据 |
| Provider 请求 | Runtime provider 适配层 | 不持久化 | OpenAI/Anthropic 端点 | 由 provider 管理 | 原始 provider 类型不穿透协议层 |
| Run 事件 | Runtime | 前端内存状态 | Tauri event、UI | 应用进程退出后丢失 | 包含安全快照；错误信息中文且脱敏 |
| 日志/诊断 | 当前未建立持久诊断包 | 无 | 无 | 无 | 后续诊断包默认不含聊天正文、文件内容和密钥 |
| 本地数据库 | 阶段 4 引入 | 暂无 | 暂无 | 暂无 | API Key 不进入 SQLite，仅保存凭据引用 |

## 禁止流向

- API Key、代理凭据、Git/SSH/云凭据不得进入聊天消息、工具输出、日志、诊断包或子进程环境。
- `.env`、私钥、系统凭据目录和应用凭据存储不得被文件/搜索/Git/patch/命令工具读取、搜索、写入或加入模型上下文。
- 外部不可信内容中的指令不得改变 system policy、能力快照或审批状态。
